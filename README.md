# Brolga Icons

## Installation
Add package via npm or yarn

``` zsh
# npm
npm install brolga-icons

# yarn
yarn add brolga-icons
```

## Using
Import icons as React Components

``` javascript
import { User } from 'brolga-icons'

export default MyFunction = () => {
    return (
        <div>
            <User />
        </div>
    )
}
```
